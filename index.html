Projects

  • Quote Keeper

    Quote Keeper is a personal project that allows me to collect quotes/inspirational sayings and store them in a single repository. A responsive layout using Rails 5, CSS 3, HTML 5, PostgreSQL which incorporates a Javascript feed from Brainy Quote with an Active Admin interface on the back-end.

  • My SportsCard

    A Rails 5 application that lets you create a personalized, digital trading card. Choose a template design, upload a photo, input your name and the card is produced using the Ruby gems Shrine, imgkit and wkhtmltoimage-binary and sent via email through SendGrid.

  • Batter's Eye

    Batter's Eye is a simple slider game where you score points by moving the slider as close as you can to the target value. The closer you get to the target value the more points you score. This project was modified from a Ray Wenderlich, Swift course.

  • War

    A classic game of War built with Swift 4 and Xcode 9. The project was an exercise from an Intro to iOS programming tutorial.
