Projects

  • Quote Keeper

    Quote Keeper is a personal project that allows me to collect quotes/inspirational sayings and store them in a single repository. A reponsive layout using Rails 5, HTML 5, PostgreSQL which incorporates a photo API from Unsplash and a Javascript feed from Brainy Quote.

  • Magic 8 Ball

    Continuing to learn Swift 4 and Xcode 9, Magic 8 Ball incorporates Swift Data Types, Variables, Arrays and includes Motion Detection.

  • War

    A classic game of War built with Swift 4 and Xcode 9. The project was an exercise from an Intro to iOS programming tutorial.

  • MyPhotoCard

    A Rails5 application that lets you create a personalized, digital trading card. Choose a template design, upload a photo, input your name and the card is produced using the Ruby gems Shrine, imgkit and wkhtmltoimage-binary and sent via email through SendGrid.

  • This Is Us Question/Answer

    A Rails5 application that requires users to SignUp/SignIn to ask, answer and upvote answers from other users on the front end. The back end includes an Administrate dashboard for site admins.

  • FreeShelf

    FreeShelf is a Rails5 application that allows a user to create, read, update and delete a list of books that incorporates Bootstrap styling and a bit of Photoshop on the front end.

Technologies
